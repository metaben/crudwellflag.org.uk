One of the first things we did in Crudwell was focus on maintenance, because we felt it had become reactive. 
We created this spreadsheet to itemise what needed doing at all ends of the village. Having one list has given conversations clarity and consistency.
A year on, we're now improving this by creating an annual schedule becauses we think this should all be done year-in, year-out.

### What software is required?

None. In Crudwell, we chose to use Google Sheets because:

- we want to share links, not files.
- it's free with a Google Account.
- it is a live editable document, there is only ever one version.
- instead of e-mailing files around all the time, we "publish" the sheet and put the link in a chat channel.
- no software is required, it works across the community on any device.
